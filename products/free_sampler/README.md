# Node-Null Free Sampler Pack v01

This is a free, shareable sampler of the Node-Null product line.

Purpose:

- Let you evaluate the writing/system style before buying.
- Provide small, safe examples (no real user data).

Included:

- One-page demos for each product (see `demos/`)
- A small subset of the Content Template Pack (see `template_pack_samples/`)

Not included:

- Full template libraries
- Full schemas/specs
- Any real run data
- Any build outputs

If you want the full kits (Personal/Team/Commercial), see the product pages.
