# Structure Template — Educational

Version: v01

## Overview

Educational content structure for explaining concepts or sharing knowledge.

## Duration

- Target: 30-60 seconds
- Minimum: 25 seconds
- Maximum: 90 seconds

## Structure

```
┌─────────────────────────────────────────┐
│ PROBLEM/QUESTION (0-8s)                 │
│ Establish what we're solving/explaining │
├─────────────────────────────────────────┤
│ EXPLANATION (8-45s)                     │
│ Deliver the answer/knowledge            │
├─────────────────────────────────────────┤
│ TAKEAWAY (45-60s)                       │
│ Summarize key point                     │
└─────────────────────────────────────────┘
```

## Section breakdown

### Problem/Question (0-8s)
- Frame what the viewer will learn
- Establish relevance
- Create knowledge gap

### Explanation (8-45s)
- Clear, structured information
- Step-by-step if process
- Examples where helpful
- Build understanding progressively

### Takeaway (45-60s)
- Summarize the key insight
- Reinforce the main point
- Optional: open loop for related content

## Timing guidelines

| Section | Start | End | Duration |
| --- | ---: | ---: | ---: |
| Problem | 0s | 8s | 8s |
| Explanation | 8s | 45s | 37s |
| Takeaway | 45s | 60s | 15s |

## Guidelines

- Assume minimal prior knowledge
- Use concrete examples
- One concept per piece
- Clear, jargon-free language

## Anti-patterns

- Talking down to viewers
- Overcomplicating simple concepts
- Multiple unrelated concepts
- No clear takeaway
