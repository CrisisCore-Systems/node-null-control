# Structure Template — Narrative

Version: v01

## Overview

Story-based content structure for engaging narrative content.

## Duration

- Target: 30-60 seconds
- Minimum: 25 seconds
- Maximum: 90 seconds

## Structure

```
┌─────────────────────────────────────────┐
│ SETUP (0-10s)                           │
│ Establish context and stakes            │
├─────────────────────────────────────────┤
│ TENSION (10-40s)                        │
│ Build conflict or complication          │
├─────────────────────────────────────────┤
│ OPEN RESOLUTION (40-60s)                │
│ Partial resolution, leave questions     │
└─────────────────────────────────────────┘
```

## Section breakdown

### Setup (0-10s)
- Establish the situation
- Introduce key elements
- Set viewer expectations
- Create initial interest

### Tension (10-40s)
- Introduce complication
- Build stakes
- Create uncertainty
- Maintain engagement

### Open Resolution (40-60s)
- Provide partial closure
- Leave some questions
- Natural stopping point
- Implicit continuation potential

## Timing guidelines

| Section | Start | End | Duration |
| --- | ---: | ---: | ---: |
| Setup | 0s | 10s | 10s |
| Tension | 10s | 40s | 30s |
| Resolution | 40s | 60s | 20s |

## Guidelines

- Story should have a point
- Tension should be natural, not manufactured
- Characters/elements should be relatable
- Resolution should be satisfying even if open

## Anti-patterns

- Stories with no point
- Manufactured drama
- Cliffhangers with no payoff
- Unrelatable or alienating content
