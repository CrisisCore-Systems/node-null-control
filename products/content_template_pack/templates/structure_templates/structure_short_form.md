# Structure Template — Short Form

Version: v01

## Overview

Standard short-form content structure optimized for 20-35 second content.

## Duration

- Target: 20-35 seconds
- Minimum: 15 seconds
- Maximum: 45 seconds

## Structure

```
┌─────────────────────────────────────────┐
│ HOOK (0-3s)                             │
│ Capture attention immediately           │
├─────────────────────────────────────────┤
│ INSIGHT (3-25s)                         │
│ Deliver core value / information        │
├─────────────────────────────────────────┤
│ OPEN LOOP (25-35s)                      │
│ End without full resolution             │
└─────────────────────────────────────────┘
```

## Section breakdown

### Hook (0-3s)
- Use hook template (H1-H4)
- Immediately engaging
- Sets topic context

### Insight (3-25s)
- Core content delivery
- Single main point
- Clear, digestible information
- Building tension or interest

### Open Loop (25-35s)
- End before full resolution
- Create desire for more
- No explicit CTA required
- Natural continuation point

## Timing guidelines

| Section | Start | End | Duration |
| --- | ---: | ---: | ---: |
| Hook | 0s | 3s | 3s |
| Insight | 3s | 25s | 22s |
| Open Loop | 25s | 35s | 10s |

## Guidelines

- One main idea only
- No tangents or secondary points
- Pacing should feel natural, not rushed
- Open loop should feel natural, not forced

## Anti-patterns

- Multiple competing ideas
- Forced cliffhangers
- Rushing through content
- Explicit "follow for part 2" CTAs
