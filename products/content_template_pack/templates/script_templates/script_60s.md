# Script Template — 60 Second

Version: v01

## Overview

Framework for 60-second (50-70s) content scripts.

## Target duration

50-70 seconds

## Script structure

```
┌─────────────────────────────────────────┐
│ HOOK (0-5s) ~15-25 words                │
│ [Hook line - use H1-H4 template]        │
├─────────────────────────────────────────┤
│ DEVELOP (5-50s) ~120-160 words          │
│ [Full development with multiple points] │
├─────────────────────────────────────────┤
│ CLOSE (50-60s) ~25-40 words             │
│ [Wrap up with insight or open question] │
└─────────────────────────────────────────┘
```

## Word count targets

| Section | Words | Speaking pace |
| --- | ---: | --- |
| Hook | 15-25 | Fast/punchy |
| Develop | 120-160 | Normal/varied |
| Close | 25-40 | Measured |
| **Total** | **160-225** | |

## Template

```markdown
## HOOK (5s)
[Your hook line here - see hook templates]

## DEVELOP (45s)
[Your main content development]

### Point 1 (15s)
- Main assertion
- Supporting detail
- Example or evidence

### Point 2 (15s)
- Second assertion
- Supporting detail
- Connection to main idea

### Point 3 (15s)
- Third assertion or synthesis
- Tie points together
- Build toward conclusion

## CLOSE (10s)
[Conclusion]
- Key takeaway
- Broader implication
- Or open question for thought
```

## Guidelines

- Full development with multiple supporting points
- Can tell a more complete story
- Still focused on one main idea
- Close should be satisfying but can be open

## Pacing notes

- Hook still fast
- Develop section has room for variation
- Can slow down for important points
- Close should feel natural, not rushed
