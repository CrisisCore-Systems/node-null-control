# Script Template — 20 Second

Version: v01

## Overview

Framework for 20-second (15-25s) content scripts.

## Target duration

15-25 seconds

## Script structure

```
┌─────────────────────────────────────────┐
│ HOOK (0-2s) ~5-15 words                 │
│ [Hook line - use H1-H4 template]        │
├─────────────────────────────────────────┤
│ CORE (2-17s) ~40-60 words               │
│ [Main content - single point]           │
├─────────────────────────────────────────┤
│ LOOP (17-20s) ~10-20 words              │
│ [Open ending - no full resolution]      │
└─────────────────────────────────────────┘
```

## Word count targets

| Section | Words | Speaking pace |
| --- | ---: | --- |
| Hook | 5-15 | Fast/punchy |
| Core | 40-60 | Normal |
| Loop | 10-20 | Slightly slower |
| **Total** | **55-95** | |

## Template

```markdown
## HOOK (2s)
[Your hook line here - see hook templates]

## CORE (15s)
[Your main content here]
- Point 1
- Point 2 (if needed)
- Supporting detail

## LOOP (3s)
[Open ending]
- Unresolved element
- Or natural pause point
```

## Guidelines

- Write for speaking, not reading
- Use short sentences
- One breath per phrase
- Practice timing with actual speech

## Pacing notes

- Fast start captures attention
- Core pacing depends on complexity
- Loop should feel natural, not rushed
