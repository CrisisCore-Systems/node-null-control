# Script Template — 30 Second

Version: v01

## Overview

Framework for 30-second (25-35s) content scripts.

## Target duration

25-35 seconds

## Script structure

```
┌─────────────────────────────────────────┐
│ HOOK (0-3s) ~10-20 words                │
│ [Hook line - use H1-H4 template]        │
├─────────────────────────────────────────┤
│ BUILD (3-23s) ~60-80 words              │
│ [Develop the idea with supporting info] │
├─────────────────────────────────────────┤
│ OPEN (23-30s) ~15-25 words              │
│ [Open ending - leaves viewer thinking]  │
└─────────────────────────────────────────┘
```

## Word count targets

| Section | Words | Speaking pace |
| --- | ---: | --- |
| Hook | 10-20 | Fast/punchy |
| Build | 60-80 | Normal |
| Open | 15-25 | Measured |
| **Total** | **85-125** | |

## Template

```markdown
## HOOK (3s)
[Your hook line here - see hook templates]

## BUILD (20s)
[Your main content development]
- Opening point
- Supporting evidence/example
- Second point (if needed)
- Transition to conclusion

## OPEN (7s)
[Open ending]
- Key insight
- Unresolved question or implication
```

## Guidelines

- More room for development than 20s
- Still one main idea, but can include support
- Build section can have 2-3 sub-points
- Open ending should feel complete but expandable

## Pacing notes

- Hook still needs to be fast
- Build section can breathe more
- Open ending should create thought, not frustration
