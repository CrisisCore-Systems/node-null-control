# Caption Template — CTA-Free

Version: v01

## Overview

Clean caption format with zero engagement bait or calls to action.

## Character count

- Target: 50-300 characters
- Maximum: 500 characters

## Structure

```
[Content summary or insight]

[Additional context if needed]

[No CTAs, no hashtags required]
```

## Template

```
{{content_summary}}

{{context_if_needed}}
```

## Examples (structure only)

```
The mechanics behind attention distribution.
```

```
Three patterns that emerged from this week's analysis.

Based on 150+ samples across platforms.
```

```
How structure affects completion rates.

A look at what the data shows.
```

## Guidelines

- Pure content summary
- No asks of any kind
- Let the content speak for itself
- Hashtags optional (0-2 if used)

## What NOT to include

- ❌ "Follow for more"
- ❌ "Like if you agree"
- ❌ "Comment your thoughts"
- ❌ "Share with someone who needs this"
- ❌ "Link in bio"
- ❌ "Turn on notifications"
- ❌ Excessive hashtags
- ❌ Questions designed to farm engagement

## Philosophy

The content is the value. If it's worth engaging with, people will engage. No manipulation required.
