## Overview

Extended caption format with context and narrative.

## Character count

- Target: 200-500 characters
- Maximum: 800 characters

## Structure

```
[Opening line - hooks reader]

[Context or background - 1-2 sentences]

[Key insight or point]

[Optional: question for thought]

[Hashtags - minimal]
```

## Template

```
{{opening_hook}}

{{context_paragraph}}

{{key_insight}}

{{thought_question}} (optional)

#{{topic_tag}} #{{category_tag}}
```

## Examples (structure only)

```
I've been studying this pattern for weeks.

The data shows something unexpected about how [topic] 
actually works. Most analysis misses this because 
[reason].

The insight: [key point].

What patterns have you noticed?

#analysis #patterns
```

## Guidelines

- Opening line should capture attention
- Context should be relevant, not filler
- Key insight should be clear
- Questions should be genuine, not engagement farming
- Hashtags minimal (2-4 max)

## Anti-patterns

- Filler content
- "Story time" without substance
- Engagement bait questions
- Excessive hashtags
- Calls to action for follows/likes
