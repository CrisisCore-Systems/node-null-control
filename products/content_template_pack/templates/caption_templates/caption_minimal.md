## Overview

Brief, algorithmic-friendly caption format.

## Character count

- Target: 50-150 characters
- Maximum: 200 characters

## Structure

```
[One-line summary or key point]

[1-3 relevant hashtags - optional]
```

## Template

```
{{content_summary}}

#{{topic_tag}} #{{category_tag}}
```

## Examples (structure only)

```
The pattern most people miss.

#patterns #signals
```

```
Why this works differently than expected.

#systems
```

## Guidelines

- One clear point
- No filler words
- Hashtags optional and minimal
- No engagement bait

## Anti-patterns

- "Link in bio"
- "Follow for more"
- Excessive hashtags
- Questions designed to farm comments
