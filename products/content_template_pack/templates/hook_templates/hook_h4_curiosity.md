Version: v01

## Structure

```
[Incomplete information that creates an information gap]
```

## Timing

- Duration: 1.5-3 seconds
- Placement: First line/frame

## Formula

```
"{{partial_reveal}} {{gap_indicator}}..."
```

or

```
"There's something about {{topic}} that {{incomplete_statement}}"
```

## Variables

- `{{partial_reveal}}`: Information that's partially shared
- `{{gap_indicator}}`: Signal that more information follows
- `{{topic}}`: Subject matter
- `{{incomplete_statement}}`: Statement that naturally continues

## Examples (structure only)

- "There's a pattern in [topic] that most people miss..."
- "The thing about [subject] is..."
- "[Topic] has a feature that [incomplete]..."

## Guidelines

- Gap must be filled in the content
- Information should be genuinely interesting
- Curiosity should be respectful, not manipulative
- Payoff should match the setup

## Anti-patterns (do NOT use)

- Bait-and-switch (gap never filled)
- Manufactured mystery with no substance
- Information gaps designed to frustrate
- Teasing content that doesn't deliver
