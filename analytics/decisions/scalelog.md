# Scale Log

Decision log for patterns and verticals that were scaled.

See [analytics/schema.md](../schema.md) for scale criteria and [docs/OPERATIONS.md](../../docs/OPERATIONS.md) for the full workflow.

---

## Format

```markdown
## YYYY-WXX

### Scaled: [Pattern/Vertical Name]

- **Reason**: [Top-quartile composite score, X% win rate, confirmed replication]
- **Samples**: N valid samples
- **Metrics**: Completion X%, Loop X%, Save+Share X%
- **Action**: [Increase output Nx | Test adjacent vertical | Expand to platform X]
- **Decision date**: YYYY-MM-DD
```
